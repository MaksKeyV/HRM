# HRM — система управления сотрудниками и зарплатами

HRM (Human Resource Management) — это веб-приложение для учета сотрудников, их должностей, отделов и истории зарплат.  
Проект написан на Python 3.10 с использованием фреймворка FastAPI для серверной части и HTML + JavaScript для веб-интерфейса.  
Для работы с базой данных используется MySQL.  

> Все операции по созданию сотрудников, добавлению отделов, должностей и управлению историей зарплат выполняются через Swagger UI.

## Развёртывание MySQL

1. Установите MySQL Server и MySQL Workbench
2. Запустите подключение MySQL Server Connections
3. Создайте базу данных для проекта (предварительно настройте пользователя, пароль и доступы)
4. Выполните SQL-команды для создания базы данных:
   `CREATE DATABASE IF NOT EXISTS <имя_базы_данных> (например: hmr_work);`
   `USE <имя_базы_данных>;`
5. В MySQL Workbench база данных появится в панели SCHEMAS слева под названием hmr_work
6. После первого запуска проекта в ней автоматически появятся таблицы:
   - departments
   - employees
   - positions
   - salary_history

## Сборка и запуск

1. После настройки MySQL Server, можно приступать к запуску проекта
2. Настройте параметры подключения к базе данных в файле `database.py`, заменив строку подключения на ваши данные:  
`DATABASE_URL = "mysql+pymysql://root:ваш_пароль@127.0.0.1:3306/hmr_work"`
3. Склонируйте репозиторий проекта
4. Установите необходимые библиотеки Python
5. Запустите MySQL сервер, если он еще не запущен
6. Запустите проект через run.bat (Этот скрипт запустит FastAPI сервер с автообновлением, откроет в браузере главную страницу и документацию Swagger UI). 


## Требования

- Операционная система: Windows  
- Язык: Python 3.10  
- СУБД: MySQL 
- Библиотеки Python: fastapi, uvicorn, sqlalchemy, pymysql, pydantic  
- Среда разработки: поддерживающая Python
- Все файлы должны располагаться в одной директории
